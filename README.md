#AI-CODE-REVIEWER
AI-powdered code review tool using GROQ

#FEATURES
-Instant AI code feedback
-Supports JS, Python, Java, C++
-Checks bugs, optimization, best practices

#TECH STACK
Frontend: React + Vite + Tailwind
Backend: Node.js +Express + GROQ
